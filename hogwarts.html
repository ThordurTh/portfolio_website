Hacked Hogwarts - Assignment

A website with a list of Hogwarts students that can be filtered by house and sorted by name.

Website

Description

This assignment was an exercise in displaying JSON data and manipulating it with vanilla Javascript. The list contains Hogwarts student's full name, their house and blood type. The list can be filtered by house and sorted and searched through by name.

Process

The website was required to have common functionality such as filtering, sorting, searching, “starring” and removing items so I began by making an activity diagram of how I would like to code it. It starts by registering all buttons for functionality and displaying the data. All of the buttons go through a chain of functions that all lead to the main “buildList” function which sends arguments to each of the functions to make sure that a filtered list is still sorted and vice versa. Then finally, it calls “displayList” and “displayStudent” to display the correct data.

First thing was to fetch and display data from two sources. The data had to be cleaned since there were inconsistencies in capitalisation and the names had to be split into first, middle and last names. Student's name and house is displayed in the list but more information such as family blood-type and gender is displayed in an overlay window when the “Details” button is clicked.

The final requirement for this assignment was to “hack the system” by injecting my own name into the list as well as mess with other data. That is done outside of the UI by calling a function in the developer tools' console called “hackTheSystem”.

Conclusion

This assignment taught me a lot about planning ahead by making an activity diagram and try to visualise how to solve certain problems. It was also helpful for learning some fundamental Javascript methods.
