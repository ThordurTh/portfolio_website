Description

This assignment was an exercise in making a dynamic website by fetching its content from a database. The theme and design was my own choice but not much time went into the design since the focus on fetching the content. The website's purpose is to display coffee shops and detailed information about each one. The user starts by selecting an area in Copenhagen and then gets a list of cafes. There they can click on each individual one and depending on what is clicked, the data will be fetched and displayed dynamically.

Process

The first thing do to was to build a database which we did on restdb.io. There we put in names, rating, pricing, address and other information found on google maps.

After coding the necessary HTML and do minimal styling with CSS, was there time to fetch the data. We used the Javascript fetch method that gets the URL for the database and returns it in JSON format. Then it's only a matter of populating each page with the right content.

Next step was to make it dynamic and fetch the right data depending on what the user clicks on. That was done by assigning each link with the right parameters so Javascript would know what to fetch and display each time.
